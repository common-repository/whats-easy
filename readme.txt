== Description ==

Whatsapp without adding contact.

Add a useful tool to you and your visitors to your site. Start a conversation on whatsapp directly without adding the contact to your phone.

To use it after activated: Add shortcode: [whats-easy] in the desired location (posts | pages | widgets)

The plugin does not yet have style edits. It pulls the style of your theme, but you can edit via CSS.

Questions, please contact us by email whats@fernandodacruz.com


== Installation ==

1. Install directly through WordPress or send the plugin to the folder `/wp-content/plugins/`.
2. Activate by admin area of WordPress.
3. Add the shortcode: [whats-easy] in the desired location (posts / pages / widgets).
4. Ready the user can enter the number and an optional message and push the button to start the conversation in the whtsapp web app or the mobile app. 


== Frequently Asked Questions ==

= Where can I enter the shortcode code? =
